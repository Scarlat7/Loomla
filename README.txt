Para utilizar o programa, o arquivo Loomla.jar e a pasta "dados" devem estar no mesmo diretório.

Para compilar os código-fontes abra os arquivos .java em um compilador de java (utilizamos o Eclipse) e crie um projeto com todos eles.
Agora, há dois passos:

1) Rodar o arquivo Trie.java, que processa os dados do arquivo fonte "dados.txt" e os coloca nas árvores TRIE e B. Não recomendamos fazer esse passo, visto que o .zip do programa já vem com esses dados processados e demora bastante (mais de meia-hora) para gerá-los novamente.

2) Exportar o .jar com a classe principal sendo a LoomlaUI.
